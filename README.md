# Paras Gold Website — How to Put This Live on GitHub (No Coding Needed)

This folder contains your full working website: Home, About, and Contact pages — built in your brand colors (beige background, deep green + gold accents), with a sliding hero banner and scroll animations.

## Step 1: Create a GitHub account (if you don't have one)
Go to https://github.com and sign up — it's free.

## Step 2: Create a new repository
1. Click the **+** icon (top right) → **New repository**
2. Name it: `paras-gold-website` (or anything you like)
3. Set it to **Public**
4. Click **Create repository**

## Step 3: Upload these files
1. On your new repository page, click **"uploading an existing file"**
2. Drag and drop ALL files and folders from this delivery (index.html, about.html, contact.html, and the `assets` folder with style.css and script.js inside it)
3. Scroll down, click **Commit changes**

> Important: keep the `assets` folder structure exactly as it is — the website depends on `assets/style.css` and `assets/script.js` being in that exact location.

## Step 4: Turn on GitHub Pages (this makes it live)
1. In your repository, click **Settings** (top menu)
2. In the left sidebar, click **Pages**
3. Under "Branch," select **main** and folder **/ (root)** → click **Save**
4. Wait 1–2 minutes, then refresh — GitHub will show you a live link like:
   `https://yourusername.github.io/paras-gold-website/`

That link is your live website. Share it, test it on your phone, and it'll keep updating automatically whenever you upload new files.
